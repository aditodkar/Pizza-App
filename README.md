# Pizza-App

# Tech Stack

1) Reactjs

2) Python Flask

3) MDBootstrap

4) Mongodb

# Installation

1) `npm install`

2) `npm start`

3) Web app runs on `localhost:3000`

4) To start backend navigate to backend folder and then `python app.py` 

5) Backend runs on `localhost:5000`

# Demo link: https://pizzaapp.netlify.com/
